# Static Site Build
This is the base repo for starting a new static build. 

### Initialize Project 
```shell
$ git clone https://github.com/nomads-every-sundays/static-build ./project-name
$ cd ./project-name
$ sudo rm -R ./.git
$ git init
$ nvm use
$ npm run init:project
```

### Scaffolding 

#### Template
